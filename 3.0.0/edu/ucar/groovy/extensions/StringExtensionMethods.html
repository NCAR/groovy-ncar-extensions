        • static String append(String self, String suffix, Integer n = 1)

          Returns a string consisting of the original string, appended by a specified string the specified number of times.

          "foo".append("bar",2) == "foobarbar"

          append(String) is append(string,1), like StringBuilder.append()

          Parameters:
          suffix - the string to append
          n - the number of times to append suffix
          Returns:
          the new string
          See Also:
          StringBuilder.append

        • static String chop(String self)

          Returns a new string that has had the last character removed from the given string. Like Perl's chop().

          Returns:
          the new shortened string

        • static String excerpt(String self, Integer length = 255)

          Returns a string no longer than length, with "..." appended if the original string was longer. Breaks the string at the last non-letter so the excerpt is readable. If length is less than three, then "..." is not appended.

          Parameters:
          length - maximum length of returned string
          Returns:
          the excerpted string
